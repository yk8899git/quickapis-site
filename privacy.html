隐私政策

QuickAPIs AI工具导航 · 最后更新：2026年5月8日

信息收集

QuickAPIs（quickapis.top）是一个AI工具导航网站。我们重视您的隐私保护。

当您访问我们的网站时，我们的服务器可能会自动记录以下信息：

  • 您的IP地址（用于统计访问量和防止滥用）
  • 访问时间与访问页面
  • 浏览器类型与操作系统
  • 推荐来源网站（Referrer）

我们使用 Google Analytics（Google Analytics 4）来分析网站流量。Google Analytics 会使用 Cookie 收集上述匿名信息，详情请参阅 Google 隐私政策。

外部链接

我们的网站包含指向第三方AI工具网站的链接。我们不对这些外部网站的隐私保护实践负责。建议您在访问任何第三方网站时查看其隐私政策。

Cookie 使用

我们使用 Cookie 来：

  • 记录您的偏好设置（如深色/浅色模式）
  • 通过 Google Analytics 了解网站使用情况
  • 确保网站安全运行

您可以通过浏览器设置拒绝 Cookie，但这可能影响部分网站功能。

信息保护

我们采取合理的技术和管理措施保护您的信息安全，防止数据遭到未经授权的访问、使用或泄露。我们的网站托管于 GitHub Pages 服务器上，遵循 GitHub 的安全标准。

第三方服务

我们使用的第三方服务：

  • Google Analytics：网站流量分析，收集匿名访问数据
  • Google Fonts：提供网站字体服务
  • GitHub Pages：网站托管服务

您的权利

根据适用法律，您可能享有以下权利：

  • 了解我们收集了关于您的哪些信息
  • 要求删除您的个人信息
  • 反对或限制我们对您信息的处理

如需行使上述权利，请通过网站的联系方式与我们联系。由于我们仅收集匿名访问数据，大部分信息在您访问结束后无法与您个人关联。

政策更新

我们可能会不时更新本隐私政策。任何更新将在本页面发布，更新日期见页面顶部"最后更新"字段。我们鼓励您定期查阅本政策。
